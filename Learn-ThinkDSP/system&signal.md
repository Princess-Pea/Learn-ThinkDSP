# 离散信号与卷积原理

## 1. 离散时间域的基本信号

### 1.1 单位样本序列（单位脉冲）

在离散时间系统中，“单位样本”与“单位脉冲”通常指同一个函数——Kronecker delta $\delta[n]$：

$$
\delta[n] = 
\begin{cases} 
1, & n = 0 \\[4pt]
0, & n \neq 0 
\end{cases}
$$

**说明**：  
- 它只在 $n=0$ 这一个采样点上有值 $1$，其余点均为 $0$。  
- 不同于连续时间单位脉冲 $\delta(t)$（无穷高、无穷窄、面积为 $1$），离散脉冲是一个确切的数值 $1$。

### 1.2 单位阶跃序列

单位阶跃序列 $u[n]$ 定义为：

$$
u[n] = 
\begin{cases} 
1, & n \ge 0 \\[4pt]
0, & n < 0 
\end{cases}
$$

即从 $n=0$ 开始，之后所有样本值均为 $1$。

---

## 2. 阶跃是脉冲的求和函数

### 2.1 累加（运行累加）视角

$$
u[n] = \sum_{m=-\infty}^{n} \delta[m]
$$

- 当 $n < 0$ 时，求和式中尚未包含 $\delta[0]$，结果为 $0$。  
- 当 $n \ge 0$ 时，$\delta[0]$ 被计入，此后每次累加都保持总和为 $1$。

### 2.2 平移脉冲叠加视角

$$
u[n] = \sum_{k=0}^{\infty} \delta[n-k]
$$

即将单位脉冲依次平移到 $n=0,1,2,\dots$ 并求和，得到从 $n=0$ 开始的全 $1$ 序列。

> **连续时间类比**：  
> 连续单位阶跃 $u(t)$ 是单位脉冲的积分：  
> $$
> u(t) = \int_{-\infty}^{t} \delta(\tau)\,d\tau
> $$

## 3. 卷积的动态理解

卷积公式（连续时间形式，便于理解）：

$$
y(t) = (x * h)(t) = \int_{-\infty}^{\infty} x(\tau) \, h(t-\tau) \, d\tau
$$

### 3.1 三阶段拆解

以 **矩形脉冲输入 $x(t)$** 与 **指数衰减冲激响应 $h(t)$** 为例：

| 信号 | 波形特征 | 物理/数学含义 |
|------|----------|----------------|
| 输入 $x(\tau)$ | 矩形脉冲（某时段为常数，之后为 $0$） | 系统受到的外部激励，有明确的起止时间 |
| 翻转平移 $h(t-\tau)$ | 反向指数波形（左高右低，带长“尾巴”） | 系统冲激响应经“翻转”与“平移”后的形态；代表系统对过去时刻的“记忆” |
| 输出 $y(t)$ | 先上升后饱和/衰减的平滑曲线 | 两个波形重叠部分的**积分面积**，即系统的总输出 |

### 3.2 为什么输入消失后输出仍然存在且衰减？

即使 $x(\tau)$ 在某个时刻之后变为 $0$，输出 $y(t)$ 往往不会立即归零，而是逐渐衰减。原因如下：

- **数学原因**：  
  只要系统冲激响应 $h(t)$ 具有“长尾巴”（如指数衰减），即使输入已为零，$h(t-\tau)$ 的尾巴仍可能与 $x(\tau)$ 的非零区域存在重叠。重叠部分的积分面积（输出）需要时间才能减小到 $0$。

- **物理原因（系统记忆/惯性）**：  
  若系统包含储能元件（电容、电感、弹簧、质量等），外部激励停止后，系统内部储存的能量不会瞬间消失，而是按自身特性缓慢释放，表现为输出的衰减“尾巴”。

> **关键点**：卷积不是简单的乘法，而是 **重叠累积**。任意时刻的输出都依赖于输入 **所有过去时刻** 的值（通过 $h(t-\tau)$ 加权）。

## 4. 物理类比：线框穿过有界磁场

| 物理概念 | 对应信号处理概念 |
|----------|------------------|
| 有界磁场区域 | 输入信号 $x(t)$（非零范围） |
| 闭合线框 | 系统的冲激响应 $h(t)$（形状决定响应方式） |
| 线框运动位移 $t$ | 时间变量 |
| 磁通量 $\Phi(t)$ | 卷积输出 $y(t)$ |

### 过程对应

- **线框进入磁场** → 重叠面积增加 → $y(t)$ 上升。  
- **线框完全在磁场内** → 重叠面积最大且恒定（若磁场均匀且线框为矩形） → $y(t)$ 进入平台或峰值区域。  
- **线框离开磁场** → 仍有部分线框在磁场内 → 重叠面积逐渐减小 → $y(t)$ 衰减（若磁场或线框形状不均匀，衰减曲线形状不同）。  

> 当冲激响应是指数型（类似线框的“尾巴”非均匀）时，离开磁场过程中磁通量（输出）平滑衰减，而不是直线下降。

---

## 5. 总结要点

1. **离散域**中，单位样本 ≡ 单位脉冲；单位阶跃 = 单位样本的累加和（从负无穷到当前时刻）。  
2. 卷积的本质是**滑动重叠积分**：输出是输入与翻转平移后的系统响应之间的重叠面积。  
3. 即使输入停止，输出仍可能持续衰减，这是由系统冲激响应的“长尾”或系统自身的能量/记忆特性造成的。  
4. 线框穿过磁场的物理类比生动诠释了“激励范围”与“响应形状”如何共同决定输出的演化过程。

> 理解这些基本关系是进一步学习 **LTI 系统分析、数字滤波、卷积神经网络** 等高级内容的重要基础。
